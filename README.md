# 乌云bug 学习App

使用方式：

1. 搭建一个后台 ( 在这里：https://github.com/sg552/bug_study_web  )
2. 导入wooyun数据库（可以到网上搜索一下）
3. 使用这个android源代码，就可以在您自己的手机上学习、查看、收藏了。
